# calculation-of-funds
Calculate the means of storing the item a certain number of days.



It is necessary to automate the calculation of the cost of container storage. 
There are a number of days of storage of all - 105 (for example) the terms of calculation are as follows:
the first 4 days - $ 4
the next 5 days - by 14
the next 5 days by 30
the next 15 days - by 36
the next every day is considered to be 50 for each day
